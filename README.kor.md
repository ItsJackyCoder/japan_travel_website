🌍 여행 웹사이트
===

[웹사이트 가기](https://explorejapan2025.netlify.app/)
--

소개
---
이 웹페이지는 일본 여행 정보를 소개하기 위해 제작되었습니다. 상호작용 기능을 구현하기보다 전체 레이아웃과 시각적 구조에 초점을 맞추었습니다.


기술 스텍
---

* 프론트엔드:`JavaScript`, `HTML5`, `CSS3`
* 프레임워크: `Bootstrap`
